## ADR Template Usage

1. Copy this template to create a new ADR: `cp docs/adr/0000-template.md docs/adr/NNNN-title.md`
2. Replace `NNNN` with the next sequential number (check existing ADRs)
3. Replace `[Title]` with a concise, descriptive title
4. Fill in each section with relevant information
5. Update the Status field as the decision progresses
6. Link related ADRs in the References section

## ADR Status Definitions

- **Proposed**: Initial draft, open for discussion
- **Accepted**: Decision has been made and approved
- **Deprecated**: Decision is no longer recommended but still valid
- **Superseded**: Decision has been replaced by a newer ADR
